# linc/test
Testing Haxe/hxcpp @:native bindings

This is a [linc](http://snowkit.github.io/linc/) library.

---

This library works with the Haxe cpp target only.

---

### Install

`haxelib git linc_test https://github.com/nslottow/linc_test.git`

### Supported platforms

- Mac
- Linux
- Windows
- Android
- iOS
